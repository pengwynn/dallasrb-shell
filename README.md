# Craft your workflow in the shell

My slides from my [talk at Dallas.rb][dallas-rb].

### Viewing the deck

First, clone the repository:

    git clone https://github.com/pengwynn/dallasrb-shell
    cd dallasrb-shell

Next install the dependencies:

    script/bootstrap

Run the slideshow:

    script/run
